# Browser Extension Template

Favourite your ChatGPT chats for quick access and pin best replies for quick branching

## Stack

TypeScript (strict), CRXJS (Vite), Chrome Manifest V3

Popup: React + Radix Themes
Content: Preact + Tailwind
Background: Effect
Lint/format: Biome

## Build

`pnpm dev` — Vite dev server with CRXJS HMR. `pnpm build` — production build to `dist/`.
